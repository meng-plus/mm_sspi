# sspi开源库
## 简介
软件SPI(Serial Peripheral Interface)在单片机开发中经常被使用，通过模块化的设计简化设计避免重复的工作量，

## 名词介绍

| 简称   | 全称                               | 中文描述    |
| ---- | -------------------------------- | ------- |
| SSPI | soft Serial Peripheral Interface | 软串行外围接口 |
|      |                                  |         |
